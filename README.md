# Login App

A Flutter login application with email and password fields.

## Features

- Email input with validation
- Password input with masking
- Form validation
- Material Design 3 UI
- Responsive layout

## Getting Started

### Prerequisites

- Flutter SDK 3.0.0 or higher
- Dart SDK

### Installation

1. Install dependencies:
```bash
flutter pub get
```

2. Run the app:
```bash
flutter run
```

## Project Structure

```
lib/
  ├── main.dart              # App entry point
  └── screens/
      └── login_screen.dart  # Login screen UI
```
